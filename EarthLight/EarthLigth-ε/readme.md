## Overview   
   
EarthLight-ε reveals that **creativity is computation**.   
   
While δ handles **Selection** (choosing from existing options), ε handles **Generation** (creating new options).   
   
---   
   
## Why ε (Epsilon)?   
   
In mathematics, ε represents "a small quantity" or "error/deviation."   
   
In evolution, **mutation** (small errors) drives all innovation.   
   
In EarthLight, **ε is the mutation engine** — the algorithm that generates novelty.   
   
---   
   
## Core Concepts   
   
### 1. Distant Element Combination   
The further apart two elements are, the more creative their combination.   
- "Astronaut × Tea Ceremony" → Still close   
- "Tea Ceremony × Battle" → Far! Interesting!   
   
### 2. The "Boring" Sensor   
```   
If idea is boring:   
    Add distant element   
    Repeat   
If idea is interesting:   
    Stop   
```   
   
### 3. Dimension Jumping (Shared with δ)   
When stuck, don't iterate — **change the dimension**.   
- Add a new axis (conflict, scale, physics)   
- Jump to a different "floor" of possibilities   
   
### 4. Pattern Recognition   
Proven structures that make things interesting:   
- Opposition (X vs Y)   
- Inflation (power creep)   
- Gap (serious × comedy)   
- Wordplay
